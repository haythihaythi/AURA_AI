# 🌟 AURA: AI-Powered Interview & Selection Operations Platform

AURA is a next-generation, AI-assisted platform designed to streamline interview and candidate selection workflows. Built with a "human-in-the-loop" philosophy, AURA acts as an intelligent interview operations assistant, reducing repetitive tasks while keeping final hiring decisions firmly in human hands.

Whether it's job recruitment, hackathons, or volunteer selection, AURA optimizes the entire pipeline—from application to final decision.

### ✨ Core Features
*   **🤖 AI Pre-screening:** Automatically analyzes CVs and applications against selection criteria to generate explainable shortlists.
*   **🎙️ Live AI Interview Assistant:** Utilizes Speech-to-Text to provide real-time transcription, highlight candidate strengths, and extract key evidence during interviews.
*   **📅 Smart Scheduling:** Generates optimized interview time slots based on interviewer availability.
*   **📊 Explainable Candidate Evaluation:** Provides data-backed recommendations and scores based on predefined rubrics.
*   **✉️ Automated Communication:** Seamlessly generates and sends tailored emails for invitations, next rounds, and professional rejections.

### 💻 Tech Stack (Frontend)
*   React & TypeScript
*   Vite (Build Tool)
*   Tailwind CSS (Glassmorphism UI Design System)
